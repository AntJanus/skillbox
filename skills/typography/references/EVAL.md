# Activation eval set

Spot-check 3–5 of these in a fresh Claude session to confirm the description triggers (and doesn't over-trigger). If activation is unreliable, rework the description in SKILL.md.

## Should trigger

1. "what font size should I use for body text?"
2. "set up a type scale for my app"
3. "this text is too small / hard to read — fix it"
4. "my headings and body don't feel balanced"
5. "what line-height for paragraphs?"
6. "set up vertical rhythm for this page"
7. "pick a font for my dashboard"
8. "pair a heading font with a body font"
9. "make this fluid/responsive with clamp()"
10. "why does my text look cramped / thin / washed out?"
11. "what's a good measure / line length for an article?"
12. "give me typography for a landing page hero"

## Should NOT trigger (adjacent skills / out of scope)

1. "what color palette should I use?"  → color-system
2. "does this color pass WCAG contrast?"  → color-system (contrast of colors, not text sizing)
3. "build a React component for my form"  → frontend-design
4. "lay out this page with a grid"  → frontend-design
5. "write copy for my landing page"  → not a typography task (content, not type)
6. "set up dark mode colors"  → color-system
